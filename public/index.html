    Introduction

    • Welcome to the OpenMRS REST API documentation! OpenMRS is a software platform and a reference application which enables the design of a customized medical records system with no programming knowledge (although medical and systems analysis knowledge is required).

    • It is a common platform upon which medical informatics efforts in developing countries can be built. The system is based on a conceptual database structure which is not dependent on the actual types of medical information required to be collected or on particular data collection forms and so can be customized for different uses.

    • This document provides High level documentation of OpenMRS APIs with code samples which are in the dark area to the right. If you find anything missing or incorrect please consider proposing revision on GitHub

    Authentication

    User

    User Overview

    User Overview A User in OpenMRS is an account that a person may use to log into the system.

    • The real-life person is represented by a Person record in OpenMRS, and a person may have more than one user account. If you want a patient to be able to view her record in OpenMRS, then you need to create a user account and link it to the patient's person record.

    Create a user

    • For convenience, the person's information can be included in order to create the corresponding person record at the same time as their user record. When creating a user record for an existing person, the existing person must only be referenced by UUID.

    • If you are not logged in to perform this action, a 401 Unauthorized status is returned.

    • The password should be minimum 8 chars long with atleast one lower and upper character alphabet along with a numeral. Some properties are not allowed to be set including name and description therefore arent included in the API usage examples.

    HTTP Request

    POST http://localhost:3000/api/v1/user

    Parameter Type Description
    name String Name of the user
    description String Description of the user
    username String username of the user
    password String password of the user
    person String person resource associated with the user
    systemId String a unique identifier assigned to each user
    roles Array[] a list of roles attributed to the user
    userProperties JSON Object A set of key value pairs. Used to store user specific data

    Update a user

    • Update a target user with given UUID, this method only modifies properties in the request. Returns a 404 Not Found status if the user does not exist. If not logged in to perform this action, a 401 Unauthorized status is returned.

    HTTP Request

    PUT http://localhost:3000/api/v1/user/:uuid

    Parameter Type Description
    name String Name of the user
    description String Description of the user
    username String username of the user
    password String password of the user
    person String person resource associated with the user
    systemId String a unique identifier assigned to each user
    roles Array[] a list of roles attributed to the user
    userProperties JSON Object A set of key value pairs. Used to store user specific data

    Update a user using its UUID

    {
    "username": "demoUserUpdated"
}

    Delete a user

    • Delete or retire a target user by its UUID. Returns a 404 Not Found status if the user does not exist. If not logged in to perform this action, a 401 Unauthorized status is returned.

    HTTP Request

    DELETE http://localhost:3000/api/v1/users/:uuid

    Parameter Description
    UUID uuid of the user to be retrieved 
    DELETE /user/:uuid

    const axios = require('axios');

axios.delete(`http://localhost:3000/api/v1/users/${uuid}`, {
  headers: {
    'Authorization': 'authKey',
    'Accept': 'application/json'
  }
})
  .then(response => {
    // Handle response
    console.log('User deleted successfully');
  })
  .catch(error => {
    // Handle error
    console.error(error);
  });

    require 'net/http'

uri = URI("http://localhost:3000/api/v1/users/#{uuid}")
http = Net::HTTP.new(uri.host, uri.port)

headers = {
  'Authorization' => 'authKey',
  'Accept' => 'application/json'
}

request = Net::HTTP::Delete.new(uri.path, headers)
response = http.request(request)

if response.code == '204'
  puts 'User deleted successfully'
else
  puts 'Failed to delete user. Status code:', response.code
end


    import requests

headers = {
  'Authorization': 'authKey',
  'Accept': 'application/json'
}

response = requests.delete(f'http://localhost:3000/api/v1/users/{uuid}', headers=headers)

if response.status_code == 204:
  print('User deleted successfully')
else:
  print('Failed to delete user. Status code:', response.status_code)
